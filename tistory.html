연산자 = 오퍼레이터

[1] 산술연산자


사칙연산. + - * / %(나머지)

[2] 관계연산자

* 크기 비교연산에 사용되는 연산자
* 비교 결과가 참(true)이면 1, 거짓(false)이면 0 * >, <, >=, <=, !=, ==

[3] 논리연산자

AND OR NOT 연산에 사용
&&(and), ||(or), ! (not)

[4] 증감연산자

* 값을 1만큼 증가 또는 감소시키는 연산자
* 증감 연산자는 반드시 변수에만 사용
* 연산자 위치에 따라 연산 순서가 다름
전위형 : a = ++cnt
후위형 : a = cnt++

[5]기타연산자

(1)비트연산자 / 비트 논리 연산자
&(and), |( or ), ~ (not) , ^(xor)
(2) 이동 shift 연산자
<< : 왼쪽 피연산자 * 2n
>> : 왼쪽 피연산자 /2 2n

* 비교 결과가 참(true)이면 1, 거짓(false)이면 0 * >, <, >=, <=, !=, ==
char = 1byte,
int , long = 4byte
double = 8byte
식별자는 제일앞에 숫자로 시작할 수 없다.
