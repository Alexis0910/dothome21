THE SKILLS WHAT I CAN DO FOR DEVELOPMENTS

ABILITY

HTML

웹표준과 웹접근성을 준수한 마크업을 할수 있으며 깔끔한 코드 정리를 통해 누가봐도 읽기 쉬운 코드로 작성이 가능합니다. HTML5 시멘틱 태그를 용도에 맞게 적재저소 사용하며 하드코딩이 가능하고 크로스 브라우징에 대해 이해하고 있습니다.

CSS

CSS의 각종 속성을 적절하게 사용하며 ir기법이나 이미지 스프라이트등과 같은 최신의 css 트렌드를 항상 주의깊게 지켜보고 있습니다. 트랜지션과 트랜스폼, 키프레임을 이용해 간단한 애니메이션을 만들수 있습니다. 그리고 항상 코드의 재활용에 대해 고민합니다.

JAVASCRIPT

슬라이드나 갤러리와 같은 플러그인의 옵션을 잘 활용하고 디자인에 맞게 적용 할 수 있습니다. 탭메뉴나 슬라이드 팝업과 같은 페이지를 구상할때 자주쓰이는 것들을 스크립트를 사용하여 구현합니다.

JQUERY

자바스크립트로 표현하기 어려운 부분을 제이쿼리 스크립트를 구현하여 좀더 다채로운 결과물을 만들 수 있습니다. 여러가지 메서드를 사용하여 스크립트에서 일어나는 이벤트나 애니메이션을 조작, 컨트롤 가능합니다.

ZEPLIN

디자인으로 작업된 결과물을 제플린과 연동하여 코딩시에 필요한 정보를 얻을 수 있고 협업시에도 유연한 소통이 가능합니다.

PHP • MYSQL

php와 mysql을 이용하여 오픈소스 데이터베이스 관리가 가능합니다. 게시판 제작이나 회원가입 등 홈페이지나 쇼핑몰 등 일반적으로 웹 개발에 널리 사용되는 소스 사용이 가능합니다.

REACT

React라는 라이브러리를 사용함으로서 상호작용이 많은 UI를 만들 때 생기는 어려움을 줄일 수 있습니다. 데이터가 변경됨에 따라 적절한 컴포넌트를 사용 가능 합니다.
